2) Multiplication of Matrices-
In linear algebra, the procedure for multiplying two matrices of size n x n is as follows:
1. Verify that the matrices are of compatible dimensions for multiplication. In order for matrix A of size n x m to be multiplied by matrix B of size p x q, the number of columns in matrix A must be equal to the number of rows in matrix B (m = p).
Otherwise-Pop up error � matrices dimensions are not compatible for multiplication.

2. Create a new matrix C of size n x q, where each element is initialized to 0.
3. For each element c(i,j) in matrix C, compute the dot product of the corresponding row i in matrix A and the corresponding column j in matrix B: 
c(i,j) = a(i,1) * b(1,j) + a(i,2) * b(2,j) + ... + a(i,n) * b(n,j)
4. Repeat steps 3 for all elements c(i,j) in matrix C.
5. The resulting matrix C is the product of matrix A and B.
It is important to keep in mind that not all matrices can be multiplied, and order of matrix also matter while multiplying.

4) Power of a Matrix �
To find the power of a matrix in linear algebra, you can use matrix multiplication. Specifically, the power of a matrix A raised to the n-th power, denoted as An, is the result of multiplying A by itself n times.
The procedure for finding the power of a matrix A to the nth power is as follows:
1. Verify that the matrix A is square, meaning that the number of rows is equal to the number of columns.
Otherwise-Pop up error �select number of rows and Column should be same(or matrix dimension mismatched or choose an square matrix)

2. Initialize a new matrix B with the same dimensions as A, and set all elements to zero.
3. Set B = A.
4. Repeat the following steps (n-1) times: 
a. Multiply B by A.
b. Set B = result of step 4a.
5. The resulting matrix B is the matrix A raised to the nth power.
For example, if A is a 2x2 matrix and we want to find A^3, we would do the following:
* A*A = B
* B*A = A^3
It is important to keep in mind that not all matrices can be raised to a power, especially non-square matrices. Additionally, some matrices that are square will not have a well-defined matrix power.

It is important to keep in mind that the transpose of a matrix is just a reordering of the elements, and does not change the underlying numbers. Therefore, transpose of a matrix is always exist, regardless of the matrix.
To find the transpose of a matrix in linear algebra, you can follow these steps:
1. Create a new matrix with the number of rows and columns swapped. For example, if the original matrix is an n x m matrix, the transpose will be an m x n matrix.
2. For each element in the original matrix, place its corresponding element in the transposed matrix. For example, the element at position (i, j) in the original matrix will be placed at position (j, i) in the transposed matrix.
3. The resulting matrix is the transpose of the original matrix.

17 ) Triangular Matrix
A triangular matrix is a type of square matrix (i.e., having an equal number of rows and columns) in which the entries below or above the main diagonal are all zero. That is, if a matrix is triangular, then all the elements below or above the main diagonal are zero, while the elements on the main diagonal and the rest of the matrix can have non-zero values. Triangular matrices are commonly used in numerical linear algebra and have several mathematical properties that make them useful in various computational tasks.
There are two types of triangular matrices: upper triangular and lower triangular. A matrix can be converted into an upper triangular matrix or a lower triangular matrix using the Gaussian elimination method or by using matrix factorization techniques such as LU decomposition.
The Gaussian elimination method involves performing row operations on the original matrix to obtain a row-reduced form that is an upper triangular matrix. The row operations include multiplying a row by a non-zero scalar, adding or subtracting two rows, and swapping two rows. The row-reduced form obtained by applying these row operations to the original matrix is the upper triangular matrix equivalent of the original matrix.
Similarly, the lower triangular matrix can be obtained by applying row operations to the original matrix to obtain a row-reduced form that is a lower triangular matrix.
It's important to note that the Gaussian elimination method may produce different results for different matrices and the results may also be sensitive to the order in which the rows are processed.

a3) Gauss Jordan Elimination Method

Often Gauss Jordan Elimination (GJE) is used to get a matrix to reduced echelon form so it is easy to solve a linear equation. Linear systems can have many variables. These systems can be solved as long as we have one unique equation/variable.

For example- two variables need two equations
Three variables need three equation to find a unique solution.
And ten variables need ten equation and so on.
In the same way 4 variables need 4 equation to find a unique solution.

Although, Gauss-Jordan elimination works on matrices of any size, they don�t have to be square. But the number of independent linear equations must not be less than number of unknown variables. However, we actually don't need. For solving �n� number of unknown variables �n� number of independent linear equations are enough. On the other hand, the given matrix needs to be square if you are using it to calculate the inverse of the matrix.

Procedure
1. Choose an n X n matrix
Otherwise- Show Pop up � please select number of rows equal to number of Columns
2. Swap the rows so that all rows with all zero entries are on the bottom. 
3. Swap the rows so that the row with the largest, leftmost nonzero entry is on top.
4. Multiply / Divide the top row by a scalar so that top row's leading entry becomes 1.
5. Add/subtract multiples of the top row to the other rows so that all other entries in the column containing the top row's leading entry are all zero.
6. Repeat steps 3-5 for the next leftmost nonzero entry until all the leading entries are 1.

TASK-3 FIND THE TYPE OF MATRIX (MATRIX STRUCTURE)-
1) Row Matrix - A row matrix is a matrix with only one row and multiple columns. It can also be considered as a 1xn matrix, where n is the number of columns.
To check if a matrix is a row matrix-  you can look at its dimensions. If the number of rows is 1 and the number of columns is greater than 1, then the matrix is a row matrix.
2) Column Matrix- A column matrix is a matrix with only one column and multiple rows. It can also be considered as a nx1 matrix, where n is the number of rows.
To check if a matrix is a column matrix, you can look at its dimensions. If the number of columns is 1 and the number of rows is greater than 1, then the matrix is a column matrix.
3) Square Matrix- A square matrix is a matrix where the number of rows and columns are equal, i.e. the number of rows is equal to the number of columns.
To check if a matrix is a square matrix, you can look at its dimensions. If the number of rows is equal to the number of columns, then the matrix is a square matrix.
4) horizontal Matrix - A matrix, in which number of rows is less than the number of columns, is called a vertical matrix.
5) Vertical Matrix - A matrix, in which number of columns is less than the number of rows, is called a vertical matrix.
6) Diagonal Matrix- A diagonal matrix is a square matrix where all the elements outside the main diagonal are zero. The main diagonal runs from the top-left to bottom-right of the matrix.
To check if a matrix is a diagonal matrix, you can check if all the elements outside the main diagonal are zero. If they are, then the matrix is a diagonal matrix.
7) Scalar Matrix- A scalar matrix is a diagonal matrix where all the elements on the main diagonal are equal. It is also known as a multiple of the identity matrix.
To check if a matrix is a scalar matrix, you can check if it is a diagonal matrix and if all the elements on the main diagonal are equal. If they are, then the matrix is a scalar matrix.
8) Null Matrix - A null matrix, also known as a zero matrix, is a matrix where all the elements are zero.
To check if a matrix is a null matrix, you can check if all the elements in the matrix are zero. If they are, then the matrix is a null matrix.
9) Lower Triangular Matrix- A lower triangular matrix is a square matrix where all the elements above the main diagonal are zero.
To check if a matrix is a lower triangular matrix, you can check if all the elements above the main diagonal are zero. If they are, then the matrix is a lower triangular matrix.
 
10) Upper Triangular Matrix - An upper triangular matrix is a square matrix where all the elements below the main diagonal are zero.
To check if a matrix is an upper triangular matrix, you can check if all the elements below the main diagonal are zero. If they are, then the matrix is an upper triangular matrix.
11) Orthogonal Matrix - An orthogonal matrix is a square matrix where the transpose of the matrix is equal to its inverse. This means that the dot product of the matrix and its transpose is equal to the identity matrix.
To check if a matrix is an orthogonal matrix, you can calculate its transpose and see if it is equal to its inverse. If it is, then the matrix is an orthogonal matrix. Another way to check this is to calculate the dot product of the matrix and its transpose, and see if it is equal to the identity matrix.
12) Singular Matrix- A singular matrix is a square matrix whose determinant is equal to zero. This means that the matrix is not invertible and its rows or columns are linearly dependent.
To check if a matrix is singular, you can calculate its determinant and see if it is equal to zero. If it is, then the matrix is a singular matrix. Another way to check this is to see if the matrix is not invertible, meaning that it has no inverse.
13) Non-Singular Matrix- A non-singular matrix is a square matrix whose determinant is not equal to zero. This means that the matrix is invertible and its rows and columns are linearly independent.
